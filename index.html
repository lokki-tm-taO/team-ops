🔥 Firebase 連携設定
手順:
1. Firebase Console でプロジェクト作成
2. Firestore Database を作成（テストモードでOK）
3. プロジェクト設定 → アプリ追加（Web）→ 設定値をコピー
4. このHTMLファイルの先頭 FIREBASE_CONFIG を書き換えて保存
5. GitHub Pages や任意のホスティングにアップロード
6. URLをチームに共有すれば完了！

Firestore セキュリティルール（開発用）:
allow read, write: if true;
設定後はリアルタイムで全メンバーのデータが同期されます。
本番運用時はFirebase Authenticationと組み合わせてセキュリティルールを設定してください。
